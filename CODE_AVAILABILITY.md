# Code availability

This repository contains the analysis code used for the manuscript
**Evolutionary diversification of the biological screw joint in weevils**.
The current workflows are located in `scripts/analysis`.

The release includes morphometric, allometric, joint-geometry, coxal-wall,
ecological and phylogenetic comparative analyses, together with the
study-specific phylogeny-preparation workflow. Final journal-figure layout,
colour adjustment, panel assembly and visual polishing scripts are excluded;
some analysis scripts still create diagnostic plots as workflow byproducts.

Study data are distributed separately as described in the manuscript. Local
absolute paths have been replaced by explicit placeholder roots or
command-line arguments so that no workstation-specific paths are published.
