# Code Availability

This repository contains the analysis code used for the manuscript.

The most current scripts are located in:

```text
scripts/analysis
```

Final journal-figure layout, colour-adjustment, panel-assembly and polishing scripts are intentionally omitted from this public code release. Some analysis scripts may still create diagnostic plots as workflow byproducts.
