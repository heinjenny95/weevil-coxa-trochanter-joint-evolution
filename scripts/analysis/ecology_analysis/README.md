# Ecology analysis

`ecology_comparative_analysis.R` merges the literature-derived ecological
coding matrix with the phylogenetic tip-level phenotype table. It produces
descriptive summaries, non-phylogenetic Wilcoxon tests, phylogenetic ANOVA and
PGLS results for the broad ecological predictors used in the manuscript.

Composite tips coded as mixed are retained descriptively but excluded from
the pairwise angiosperm-versus-gymnosperm host-lineage tests.
